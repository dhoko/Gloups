# Gloups

## Installation

Run `npm install`

## Create an application

`./src/bin/gloups make`

### Exemple

I've cloned the repository in `/home/dhoko/dev/Gloups/`, so the binary is in `/home/dhoko/dev/Gloups/src/bin/`.

To setup an app (*in any folder*), run `./home/dhoko/dev/Gloups/src/bin/gloups make`
